## 基于知识图谱的金融风险事件分析系统

### 项目简介

本项目是一个基于知识图谱的金融风险事件分析系统，旨在帮助分析金融风险事件之间的内在联系和潜在规律。该系统结合了前沿的图数据库技术Neo4j与强大的数据分析可视化工具ECharts，以提供一个直观、高效的平台。


### 主要功能

- 登录注册功能
- 知识图谱展示与查询
- 数据初始化与导入


### 技术架构

**后端:**
- Python语言
- Django框架
- Py2neo（Neo4j图数据库驱动）


**前端:**
- HTML/CSS/JavaScript
- ECharts可视化库


### 使用说明

1. 下载并运行项目
2. 登录系统，使用标准的用户名和密码
3. 探索知识图谱，理解金融风险事件之间的联系
4. 使用查询功能快速定位感兴趣的风险事件或实体
